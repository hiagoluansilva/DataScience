# DataScience

Notebooks Jupyter de estudo em **Ciência de Dados** com Python — da programação básica à regressão linear.

## Conteúdo

| Aula | Notebook | Tópico |
|------|----------|--------|
| Aula 1 | `Programando_com_Python.ipynb` | Introdução ao Python para Data Science |
| Aula 2 | `Aula_2.ipynb` | Manipulação de dados e análise exploratória |
| Aula 3 | `ExemploCorrelacao.ipynb` | Correlação entre variáveis |
| Aula 4 | `ExemploCorrelacao_Regressao.ipynb` | Regressão linear |

## Conceitos abordados

- Sintaxe Python, tipos de dados, estruturas de controle
- Leitura e manipulação de datasets (CSV/TXT)
- Análise exploratória de dados (EDA)
- Correlação de Pearson
- Regressão linear simples e múltipla
- Visualização com Matplotlib/Seaborn

## Requisitos

```bash
pip install jupyter numpy pandas matplotlib seaborn scikit-learn
```

## Como usar

```bash
jupyter notebook
```

Abra o notebook desejado e execute as células em sequência.
